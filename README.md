# QA Interview Guide

Структурированный гайд по подготовке к собеседованию на позицию **QA Automation Java (Middle+ / Senior)**.

## О проекте

Покрывает ~95% типовых тем интервью в российских продуктовых компаниях (с упором на fintech).  
Содержит ~410 вопросов с развёрнутыми ответами, примерами кода и mermaid-диаграммами.

**Стек:** Java · JUnit 5 · Maven · Playwright · RestAssured · Allure · Spring · Docker · SQL · CI/CD
